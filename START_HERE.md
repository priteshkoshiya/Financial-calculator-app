## 🎨 Customization

### Change Site Name
Edit `app/layout.tsx`:
```tsx
title: 'YourSiteName | Financial Calculators'
```

### Change Colors
Edit `app/globals.css`:
```css
--primary: oklch(0.45 0.25 256); /* Your brand color */
--secondary: oklch(0.55 0.18 182);
--accent: oklch(0.65 0.22 45);
```

### Change Contact Email
Edit `app/contact/page.tsx`:
```tsx
support@youremail.com
```

**For complete customization guide → See CUSTOMIZATION_GUIDE.md**
